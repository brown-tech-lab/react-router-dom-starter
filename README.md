## Hello!
## This is a React Router Starter

----------------------------------
## How to use: 

```
npm install
npm start
```
- That will run a dev server on port 8080
- Open your browser and surf to http://localhost:8080
- Exit dev server by pressing CTRL + C

----------------------------------
## How to build:
```
npm run build
```
You can find built files in `./dist` directory
